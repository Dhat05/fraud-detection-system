# Bank Fraud Detection System

## Overview
A Python-based fraud detection system that analyzes banking transactions and identifies suspicious transactions.

## Features
- Reads transaction data from CSV
- Detects fraudulent transactions
- Generates fraud reports
- Logs transaction analysis
- Built on Ubuntu Linux
